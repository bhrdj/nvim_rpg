NVIM DUNGEON CURRICULUM
=======================

LESSON PLAN
-----------
Floor 1: Movement
  1.1  Cardinal Movement    hjkl
  1.2  Word Navigation      w b e W B E
  1.3  Line Edges           0 ^ $
  1.4  Count Modifiers      {number}{motion}
  1.5  File Navigation      gg G {n}G

Floor 2: Operators
  2.1  Core Operators       d c y
  2.2  Text Objects         iw aw i" a" i( a( i{ a{
  2.3  Line Operations      dd cc yy D C
  2.4  Paste Commands       p P

Floor 3: Search
  3.1  Pattern Search       / ? n N
  3.2  Cursor Search        * #
  3.3  Line Find            f t F T
  3.4  Find Repeat          ; ,

Floor 4: Workspace
  4.1  Split Windows        :sp :vs Ctrl-w s/v
  4.2  Window Navigation    Ctrl-w h/j/k/l
  4.3  Terminal Mode        :terminal, Ctrl-\ Ctrl-n (exit insert)
  4.4  Buffer Commands      :e :bn :bp :ls :b{n} :enew
  4.5  File Tree            (plugin-dependent: nvim-tree, neo-tree)

Floor 5: Advanced
  5.1  Visual Mode          v V Ctrl-v
  5.2  Undo Redo            u Ctrl-r
  5.3  Macro Recording      q @
  5.4  Position Marks       m ' `
  5.5  Register Power       " @

INVENTORY (Mastered Skills)
---------------------------
hjkl, wbe, 0^$, counts, d/c/y, text objects, /?nN :noh, *#, ft;,
:sp/:vs, Ctrl-w hjkl, :terminal, Ctrl-\ Ctrl-n
v, V, "a-z registers, Alt+v terminal paste, :enew scratch buffer
