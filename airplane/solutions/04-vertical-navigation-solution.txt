VERTICAL NAVIGATION - SOLUTION
==============================

| Goal              | Key(s)          |
|-------------------|-----------------|
| Top of file       | gg              |
| Bottom of file    | G               |
| Go to line 42     | 42G  or  :42    |
| Half-page down    | Ctrl-d          |
| Half-page up      | Ctrl-u          |
| Full page down    | Ctrl-f          |
| Full page up      | Ctrl-b          |

Mnemonic: d=down, u=up, f=forward, b=back
