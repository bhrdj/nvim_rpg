WORD HOPPING
============

Fill in the key for each motion:

| Goal                             | Key |
|----------------------------------|-----|
| Next word start                  | _w_ |
| Previous word start              | _b_ |
| End of current word              | _e_ |
| Next WORD (whitespace-delimited) | _W_ |
| Previous WORD                    | _B_ |
| End of WORD                      | _E_ |
